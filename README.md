# Bagels Game:

--.-- **ESPAÑOL:** --.--

   Bagels se trata de un juego clásico de adivinanza numérica en el mundo de la programación. ¡Pero ahora con implementación Docker! Para que puedas probar la correcta instalación de Docker Compose de una manera diferente y divertida :D

# Prerrequisitos:

  Para poder jugar, necesitarás tener instalado "Docker" y "Docker Compose" en tu sistema. Puedes en la terminal verificar si ambos están instalados con estos comandos:

   docker --version
   docker-compose --version

# INSTRUCCIONES DE EJECUCIÓN:

   1. **Clona el repo:** Abre tu terminal y ejecuta:

        git clone https://github.com/tu-usuario/Docker-Bagels.git
        cd Docker-Bagels

   2. **Asegurate de ubicarte en la carpeta del juego** De lo contrario tendrás errores con el siguiente paso.
    
   3. **Ahora, ejecuta el siguiente comando en la terminal:**

        docker-compose run --rm bagels-game
